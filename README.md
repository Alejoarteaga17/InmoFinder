# InmoFinder
Hi there, this is the gitHub repo to our website project from the P1 class. 
## Usage instructions

1. **Clone the GitHub repository**
	```pwsh
	git clone https://github.com/Alejoarteaga17/InmoFinder
	cd InmoFinder
	```

2. **Install dependencies**
	```pwsh
	pip install -r requirements.txt
	```

3. **Run the project**
- Will generate the embeddings so this might take a bit.
	```pwsh
	python manage.py migrate
	python manage.py runserver
	```

Go to [http://localhost:8000](http://localhost:8000) and use the app
